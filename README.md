# PIXEL AGENCY — AI команда создаёт сайты

AI-агентство, где 9 агентов создают готовый лендинг за минуты. Бесплатно.

## 🚀 О проекте

Интерактивное веб-приложение, демонстрирующее работу виртуального AI-агентства по созданию лендингов. В приложении реализована симуляция офиса с агентами, каждый из которых отвечает за свою задачу в процессе создания сайта.

## ✨ Особенности

- **9 AI-агентов** с различными ролями:
  - Project Manager
  - Researcher
  - Copywriter
  - Designer
  - Frontend Developer
  - Backend Developer
  - QA Engineer
  - DevOps
  - CEO

- **Интерактивный офис** с визуализацией рабочих мест агентов
- **Панель управления** для настройки параметров проекта
- **Современный UI/UX** с неоновым дизайном и анимациями
- **Адаптивный дизайн** для различных устройств

## 🛠 Технологии

- HTML5
- CSS3 (с CSS-переменными и градиентами)
- JavaScript (Vanilla)
- Canvas API для рендеринга офиса

## 🚀 Запуск

Просто откройте файл `index.html` в любом современном браузере:

```bash
# Локально
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux

# Или через локальный сервер
python -m http.server 8000
# Затем откройте http://localhost:8000
```

## 🎮 Использование

1. Откройте приложение в браузере
2. Настройте параметры проекта в правой панели:
   - Название компании
   - Тип бизнеса
   - Цветовую схему
   - Дополнительные опции
3. Наблюдайте за работой AI-агентов в виртуальном офисе
4. Получите готовый лендинг
